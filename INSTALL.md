# MOONAI Chat Widget — Multi-Store Setup Guide

## Architecture

```
[Shopify Store]  →  theme.liquid snippet  →  [Vercel Backend]  →  Claude AI
                         ↑                          ↑
                  Edit in Shopify              env vars per store
                  Theme Editor                (color, agent, catalog)
```

One Vercel deployment per store. Each store gets its own env vars — no code changes ever needed.

---

## Customize UI in Shopify Theme Editor

Go to: **Shopify Admin → Online Store → Themes → Customize**

In the left sidebar, click **"App embeds"** or find the **"MOONAI Chat Widget"** section.

You can edit:

| Setting | What it changes |
|---|---|
| **Brand Color** | Launcher button, sent message bubbles, CTA buttons |
| **Background Color** | Chat window background |
| **Agent Name** | Name shown in the header (e.g. "Armie") |
| **Agent Initials** | Fallback avatar text (e.g. "LA") |
| **Profile Image URL** | Agent avatar photo |
| **Widget Position** | Bottom-right or bottom-left |

To change these settings in the widget snippet directly, find the `IMSG` config block at the top of the `<script>` section in `chat-widget-v2.html` and update:

```js
const IMSG = {
  agentName:    'Armie',
  agentInitials:'LA',
  agentAvatar:  '',            // URL to profile image (leave blank for initials)
  brandColor:   '#c9a84c',     // Gold for Limited Armor
  bgColor:      '#0a0a0a',     // Near-black background
  apiUrl:       'https://imessage-chat-widget.vercel.app',
  position:     'right',
};
```

## Add a New Store (5 minutes)

### Step 1 — Run the setup script
```bash
python3 setup_store.py
```
Enter your store details when prompted. The script will:
- Connect to Shopify and pull all products, collections, policies
- Generate a `.env.{storename}` file with all required variables
- Output the Shopify snippet pre-configured for that store

### Step 2 — Create a new Vercel project
Duplicate the `imessage-chat-widget` project in the Vercel dashboard, or:
```bash
vercel --token YOUR_TOKEN --name {store-name}-chat
```

### Step 3 — Set environment variables
Copy everything from `.env.{storename}` into Vercel's Environment Variables panel, plus:
```
ANTHROPIC_API_KEY=sk-ant-...
SHOPIFY_ACCESS_TOKEN=shpat_...
SHOPIFY_SHOP_DOMAIN=yourstore.myshopify.com
```

### Step 4 — Deploy
Push to GitHub — Vercel auto-deploys in ~20 seconds.

### Step 5 — Install on Shopify
1. Update `apiUrl` in the widget `IMSG` config block to your new Vercel URL
2. In Shopify Admin → **Online Store → Themes → Edit code → `layout/theme.liquid`**
3. Paste the widget snippet just before `</body>` → Save

## Sync Catalog (when you add new products)

Re-run the setup script in sync-only mode:
```bash
SHOPIFY_ACCESS_TOKEN=shpat_... SHOPIFY_SHOP_DOMAIN=yourstore.myshopify.com python3 setup_store.py --sync-only
```
Then push to GitHub — Vercel redeploys in ~20 seconds with the updated catalog.

## Proactive Trigger Logic (Rep AI-style)

The widget fires a proactive message automatically based on:

| Trigger | Timing |
|---|---|
| Time on page (first visit) | 8 seconds |
| Time on page (return visit) | 4 seconds |
| Exit intent (mouse leaves top) | Immediate |
| Scroll depth (50% of page) | Immediate |
| Idle (no mouse movement) | 45 seconds |
| Cart page | 5 seconds |

The proactive message is generated by Claude in real-time based on the current page type (homepage, product, collection, cart) and the actual product title if on a product page.
